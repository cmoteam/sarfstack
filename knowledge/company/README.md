# Company Knowledge — 企業固有情報

このディレクトリには、あなたの企業/プロダクト固有の情報を格納します。
CMObotの各エージェントは、この情報をもとに施策のレビューや提案を行います。

## セットアップ手順

以下のファイルを、あなたの企業の情報で埋めてください:

1. **company-overview.md** — 事業概要、ミッション、主要プロダクト
2. **icp.md** — 理想的な顧客像（Ideal Customer Profile）
3. **positioning.md** — 市場ポジショニング、競合との差別化
4. **brand-guidelines.md** — ブランドガイドライン、トーン&マナー
5. **competitors.md** — 競合情報、競合との比較

## 注意事項

- 各ファイルにはテンプレートが用意されています。`[TODO]` マーカーを実際の情報で置き換えてください。
- 情報が不十分でもCMObotは動作しますが、レビューの精度が下がります。
- 機密情報は含めても構いませんが、このリポジトリのアクセス権限に注意してください。
- 定期的に情報を更新してください（特に競合情報とポジショニング）。
