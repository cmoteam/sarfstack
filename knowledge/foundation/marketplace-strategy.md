# Marketplace Strategy — EC/マーケットプレイス戦略

## マーケットプレイスの基本原則

### 勝利の方程式
```
アクセス数 × CVR > 競合 = 競合優位性
```

- **アクセス数** = マーケティング力（SEO対策、各種広告の運用力）— 影響度20%
- **CVR** = 商品力（機能、価格、デザイン）— 影響度80%

商品力がCVRの圧倒的な決定要因。マーケティングだけでは勝てない。

### ユーザーの購入導線（Amazon事例）
```
検索 60% ── 通常検索 + 広告経由
カテゴリ/特集 30%
他の商品ページ 10%
```

**検索を抑えることが最重要。** 検索KW上位を死守し続けること = 必要出荷数を出し続けること。

## 3秒ルール

- ページを訪れた顧客は**3秒以内**に購入の意思決定をする
- 約60%のユーザーがファーストビュー内で離脱or購入を決定
- ファーストビューで伝えるべき要素:
  1. 商品画像
  2. タイトル
  3. レビュー
  4. ベストセラーマーク
  5. 価格（二重価格）
  6. プロモーション
  7. バリエーション
