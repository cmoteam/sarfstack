# Latest Knowledge — 最新情報

このディレクトリには、頻繁に更新される揮発性の情報を格納します。
プラットフォームの仕様変更、業界トレンド、直近のパフォーマンスデータなど。

## 更新頻度の目安

- **platform-updates.md** — プラットフォーム仕様変更時に随時更新
- **industry-trends.md** — 月次で更新推奨
- **performance-data.md** — 週次で更新推奨

## 使い方

各スキル（特に実行系: SEO, Ads, Content）がこのディレクトリを参照します。
情報が古いと施策の精度が下がるので、定期的な更新を心がけてください。

`/weekly-retro` ワークフローの中で、最新情報の更新もリマインドされます。
