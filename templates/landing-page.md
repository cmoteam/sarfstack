# Landing Page Template — LP構成テンプレート

## Standard LP Structure

### Section 1: Hero（ヒーロー）
```
[ヘッドライン: 顧客のベネフィットを1文で]
[サブヘッドライン: 具体的な説明を1-2文で]

[CTA Button: 行動を促す具体的なテキスト]

[ヒーロー画像/動画: プロダクトの価値を視覚的に伝える]

[信頼要素: 「XX社が導入」「XX%の改善実績」等]
```

### Section 2: Problem（課題提示）
```
[見出し: ターゲットの課題をそのまま言語化]

- 課題1: [具体的なペイン]
- 課題2: [具体的なペイン]  
- 課題3: [具体的なペイン]

[放置した場合のリスクを1-2文で]
```

### Section 3: Solution（解決策）
```
[見出し: 解決策を端的に]

[プロダクトがどう課題を解決するか、3-5のポイントで]

各ポイント:
- [Feature] → [Benefit]（機能ではなくベネフィットで語る）
```

### Section 4: Social Proof（社会的証明）
```
[見出し: 「XXX社が選んだ理由」等]

[テスティモニアル × 2-3]
各テスティモニアル:
- 「引用文」
- 名前、役職、企業名
- （可能であれば顔写真）

[導入企業ロゴ × 5-8]

[実績数字: 「導入XX社」「満足度XX%」等]
```

### Section 5: How it Works（使い方）
```
[見出し: 「3ステップで始められます」等]

Step 1: [アクション] → [結果]
Step 2: [アクション] → [結果]
Step 3: [アクション] → [結果]
```

### Section 6: Pricing（料金）※該当する場合
```
[プラン比較表]
[最も人気のプランをハイライト]
[各プランのCTA]
```

### Section 7: FAQ
```
Q1: [最もよくある質問]
A1: [簡潔な回答]

Q2-Q5: [追加のFAQ]
```

### Section 8: Final CTA
```
[ヘッドラインの繰り返しまたはバリエーション]
[最後のCTA: ヒーローと同じボタン]
[リスクリバーサル: 「無料トライアル」「返金保証」等]
```

## CTA Button Text Examples

| Type | Weak | Strong |
|------|------|--------|
| 無料トライアル | 登録 | 無料で始める |
| 資料請求 | 送信 | 資料を無料ダウンロード |
| お問い合わせ | 送信 | 専門家に相談する |
| 購入 | 購入 | 今すぐ手に入れる |
