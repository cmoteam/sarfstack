# Social Media Templates — SNSコンテンツテンプレート

## X（Twitter）

### Type 1: 知見共有
```
[業界のインサイト or 学び]

具体的には:

1. [ポイント1]
2. [ポイント2]
3. [ポイント3]

[まとめの1文]
```

### Type 2: データ共有
```
[衝撃的な数字や統計] って知ってましたか？

[その数字の文脈]
[自分たちの経験からの解釈]
[読者へのアクション提案]
```

### Type 3: Before/After
```
Before: [問題のある状態]
After: [改善された状態]

変えたのは[たった1つのこと]:
[具体的な施策]

[結果の数字]
```

### Type 4: スレッド（長文）
```
Tweet 1/N:
[フックとなる導入。最も強い主張 or 意外な事実]

Tweet 2/N:
[背景の説明]

Tweet 3-N:
[本論を展開]

Tweet N/N:
[まとめ + CTA]
```

## LinkedIn

### Type 1: ストーリー型
```
[フック: 意外な結果 or 失敗から始める]

↓

[背景の説明: 何をしようとしていたか]

↓

[転機: 何が変わったか]

↓

[学び: 汎用的なインサイト]

↓

[CTA: あなたの経験は？]

#hashtag1 #hashtag2 #hashtag3
```

### Type 2: リスト型
```
[テーマ]について、[数字]年やってきてわかった[数字]つのこと:

1️⃣ [ポイント1]
→ [1文の解説]

2️⃣ [ポイント2]
→ [1文の解説]

3️⃣ [ポイント3]
→ [1文の解説]

最も重要なのは[最重要ポイント]。

[CTA]
```

## Instagram

### Type 1: カルーセル（教育系）
```
Slide 1（表紙）: [キャッチーなタイトル]
Slide 2: [課題の提示]
Slide 3-8: [各ポイントを1スライド1メッセージで]
Slide 9: [まとめ]
Slide 10: [CTA: 保存してね / フォローしてね]

キャプション:
[スライドの要約 + 補足情報]
.
.
.
[ハッシュタグ × 20-30]
```

### Type 2: Reels スクリプト
```
Hook（0-3秒）: [注意を引く最初の1文]
Body（3-30秒）: [メインコンテンツ]
CTA（最後）: [フォロー/コメント/保存 の促し]

キャプション: [要約 + ハッシュタグ]
```

## 投稿頻度の目安

| Platform | 最低頻度 | 理想頻度 | ベストタイム（日本） |
|----------|---------|---------|------------------|
| X | 1回/日 | 3-5回/日 | 7-8時, 12時, 18-21時 |
| LinkedIn | 2回/週 | 5回/週 | 8時, 12時 |
| Instagram | 3回/週 | 5-7回/週 | 12時, 18-21時 |
| note | 1回/週 | 2-3回/週 | 12時, 20時 |
