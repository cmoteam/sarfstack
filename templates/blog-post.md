# Blog Post Template — ブログ記事テンプレート

## 記事構成テンプレート

### Type 1: How-to（ハウツー記事）

```markdown
# [数字]つの方法で[ベネフィット]を実現する方法

## この記事でわかること
- [ポイント1]
- [ポイント2]
- [ポイント3]

## [課題/テーマ]とは？
[読者が抱える課題を言語化。共感を得る導入]

## [方法1]: [具体的なアクション]
[手順の説明]
[具体例やスクリーンショット]
[期待できる効果]

## [方法2]: [具体的なアクション]
...

## [方法3]: [具体的なアクション]
...

## まとめ
[要点の振り返り]
[最初の一歩として何をすべきか]

## [CTA]
[記事のテーマに関連するCTA]
```

### Type 2: Case Study（事例紹介）

```markdown
# [企業名]が[Product]で[成果]を達成した方法

## 企業プロフィール
- 業種: ...
- 規模: ...
- 課題: ...

## Before: 導入前の状況
[具体的な課題と数字]

## 施策: 何をしたか
[導入の経緯]
[具体的な活用方法]
[工夫したポイント]

## After: 導入後の成果
[Before → After の数字比較]
[定性的な変化]

## 担当者の声
「[テスティモニアル]」

## まとめ: 成功のポイント
1. ...
2. ...
3. ...

## [CTA]
```

### Type 3: Comparison（比較記事）

```markdown
# [カテゴリ]おすすめ[数字]選｜[年]年最新比較

## [カテゴリ]とは？
[カテゴリの説明]
[選ぶ際のポイント]

## 比較表
| サービス | 特徴 | 料金 | おすすめ度 |
|----------|------|------|-----------|
| ... | ... | ... | ★★★★★ |

## 1. [サービス名]
[特徴・メリット・デメリット]

## 2. [サービス名]
...

## 目的別おすすめ
- [目的A]なら → [サービス]
- [目的B]なら → [サービス]

## まとめ
[選び方の要約]

## [CTA]
```

## SEO Meta Template

```
Title: [30-60文字。KWを前方に含む。CTR意識]
Description: [120-160文字。記事の要約 + 読む理由 + CTA的要素]
```

## 記事品質チェックリスト

- [ ] タイトルに具体的な数字が入っている
- [ ] 導入で読者の課題に共感している
- [ ] 見出しだけで記事の概要がわかる
- [ ] 具体的なデータ/事例が含まれている
- [ ] 行動可能なアドバイスがある
- [ ] CTAが記事のテーマと関連している
- [ ] 1000文字以上（SEO最低ライン）
