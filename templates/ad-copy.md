# Ad Copy Templates — 広告コピーテンプレート

## Google Search Ads（レスポンシブ検索広告）

### Headlines（見出し: 最大30文字 × 15本）

**パターン1: ベネフィット訴求**
```
H1: [ベネフィット] を実現
H2: [数字] の実績
H3: まずは無料で試す
```

**パターン2: 課題訴求**
```
H1: [課題] でお困りですか？
H2: [解決策] で解決
H3: [数字]%のコスト削減
```

**パターン3: 権威性訴求**
```
H1: [業界]No.1の[カテゴリ]
H2: [数字]社が導入
H3: 導入事例を見る
```

### Descriptions（説明文: 最大90文字 × 4本）
```
D1: [課題]を[解決策]で解決。[具体的なベネフィット]。まずは無料トライアルから。
D2: [数字]社の導入実績。[ベネフィット1]、[ベネフィット2]を実現。お問い合わせはこちら。
```

## Meta Ads（Facebook / Instagram）

### Primary Text（本文: 125文字推奨）

**パターン1: PAS**
```
[Problem] に悩んでいませんか？
放置すると [Agitate] になります。
[Solution] なら [Benefit]。
▶ 詳しくはこちら
```

**パターン2: Social Proof**
```
[数字]社が選んだ [Product]。
「[テスティモニアル1文]」
あなたも [CTA]
```

### Headline（見出し: 40文字推奨）
```
[ベネフィット] | [CTA]
```

## X Ads（Twitter）

### ツイート広告（280文字以内）

**パターン1: 知見共有型**
```
[業界の課題/トレンド] について、[数字]社のデータを分析してわかったこと：

1. [知見1]
2. [知見2]  
3. [知見3]

詳しいレポートはこちら👇
[URL]
```

**パターン2: 事例紹介型**
```
[企業名]様が [Product] を導入して [期間]。

結果：
✅ [成果1]
✅ [成果2]
✅ [成果3]

事例の詳細はこちら
[URL]
```
