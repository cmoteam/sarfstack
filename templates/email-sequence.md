# Email Sequence Templates — メールシーケンステンプレート

## Welcome Series（ウェルカムシリーズ: 新規登録後）

### Email 1: Welcome（登録直後）
```
件名: [Company]へようこそ！最初にやるべき1つのこと
---
[名前]さん、

ご登録ありがとうございます！

[Product]を最大限活用するために、まずは[最初のアクション]を試してみてください。

[1つだけの明確なCTA]

何かお困りの際は、いつでもご連絡ください。

[署名]
```

### Email 2: Value（+1日）
```
件名: [ベネフィット]を実現する方法
---
[具体的なTipsや使い方を1つ紹介]
[事例やデータを添えて価値を示す]
[CTA: 試してみる]
```

### Email 3: Social Proof（+3日）
```
件名: [企業名]はこうして[成果]を達成しました
---
[顧客事例を1つ紹介]
[Before → After の変化を具体的に]
[CTA: 自分も始める]
```

### Email 4: Engagement（+7日）
```
件名: [名前]さん、順調ですか？
---
[利用状況に応じたメッセージ]
[活用Tipsの追加提供]
[CTA: 次のステップ]
```

## Nurture Sequence（ナーチャリング: リード向け）

### Email 1: Education（課題の言語化）
```
件名: [業界]で[課題]に直面していませんか？
---
[課題を具体的に描写]
[課題を放置した場合の影響]
[解決策の方向性を示す（売り込みなし）]
[CTA: 詳しい記事を読む]
```

### Email 2: Solution（解決策の提示）
```
件名: [課題]を解決する3つのアプローチ
---
[アプローチ1: 一般的な方法]
[アプローチ2: 一般的な方法]
[アプローチ3: 自社プロダクトが強い方法]
[CTA: 比較資料をダウンロード]
```

### Email 3: Proof（証拠の提示）
```
件名: [数字]%の改善を実現した方法
---
[ケーススタディの要約]
[具体的な数字と成果]
[CTA: 詳細な事例を読む]
```

### Email 4: Offer（オファー）
```
件名: [期間限定/特別] のご案内
---
[オファーの内容]
[なぜ今なのかの理由]
[CTA: 今すぐ始める]
```

## 件名のベストプラクティス

| テクニック | 例 |
|-----------|-----|
| 数字を使う | 「CVRを3倍にした5つの方法」 |
| 疑問形 | 「なぜ87%の企業はここで失敗するのか？」 |
| パーソナライズ | 「[名前]さんへの特別なご提案」 |
| 緊急性 | 「本日23:59まで」 |
| 好奇心 | 「あなたが見落としている1つのこと」 |

**避けるべき件名**:
- 全角大文字の乱用
- 過度な煽り（「！！！」の連続）
- 誤解を招く件名（内容と一致しない）
